App Router
The App Router is a file-system based router that uses React's latest features
such as Server Components, Suspense, and Server Functions.
Learn the fundamentals of building an App Router project, from installation to
layouts, navigation, server and client components.

Pre-requisite knowledge
Next.js documentation assumes some familiarity with web development. It helps to
be comfortable with HTML, CSS, JavaScript, and React. If you're new to React or
need a refresher, the React Foundations course
and the Next.js Foundations course are
good starting points.
